# Sales_dashboard
A simple Shiny dashboard that visualizes monthly sales trends and product performance.
Users can upload their own Excel files and can view the default dataset.
 **Live App:** [View it here](https://abolade-z-adewale.shinyapps.io/Shiny_prep/)

## Features
- Upload Excel file (.xlsx or .xls)
- Interactive filters (Region, Product)
- Value boxes, bar charts, and line trends

## How to Run
1. Clone the repository.
2. Open `app.R` in RStudio.
3. Run `shiny::runApp()`.
