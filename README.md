# Sales_dashboard
A simple Shiny dashboard that visualizes sales and profit data by region and product.
Users can upload their own Excel files and can view the default dataset.

## Features
- Upload Excel file (.xlsx or .xls)
- Interactive filters (Region, Product)
- Value boxes, bar charts, and line trends

## How to Run
1. Clone the repository.
2. Open `app.R` in RStudio.
3. Run `shiny::runApp()`.
